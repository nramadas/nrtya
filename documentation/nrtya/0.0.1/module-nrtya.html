Module: nrtya

(require("nrtya"))(fn) → {function}

Decorates a function. The new decorated function will execute asynchronously on a separate thread using a WebWorker. Calling the decorated function will return a Promise that will resolve with the results.

This function can only decorate pure functions. Functions that rely on closures, side-effects, and other non-pure phenominon will not work.
